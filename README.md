# Welcome to Your First GitHub Project! 🎉

You made it — this is your very first step into the world of coding and collaboration. This repository (or "repo" for short) is a hands-on tutorial that will walk you through everything you need to get started with GitHub, VS Code, and deploying your own website.

By the end, you'll have a live personal portfolio on the internet. For real.

![GitHub Pages Deploy](https://img.shields.io/badge/GitHub%20Pages-Deploy-blue?logo=github)

---

## What You'll Learn

- **GitHub** — How to create an account, make a repo, and manage your code
- **Codespaces** — A full coding environment right in your browser (no install needed)
- **VS Code** — The code editor used by millions of professional developers
- **Git** — The version control system that tracks every change you make
- **GitHub Pages** — Free website hosting, straight from your repo
