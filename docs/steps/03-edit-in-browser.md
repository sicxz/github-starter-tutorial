# Step 3: Make Your First Edit in GitHub

## Goal
Learn the fastest edit loop by changing files directly in GitHub.

## Instructions
1. Open `index.html` in your repo.
2. Click the pencil icon to edit.
3. Change your name, title, or intro text.
4. Commit the change directly to your `main` branch.

## Checklist
- [ ] I edited `index.html` in the browser.
- [ ] I created my first commit.
- [ ] I can see the update in the file history.
