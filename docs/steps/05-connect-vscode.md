# Step 5: Connect Codespaces to Local VS Code

## Goal
Open your codespace in local VS Code to get the full desktop workflow.

## Instructions
1. Install VS Code on your computer.
2. Install the GitHub Codespaces extension when prompted.
3. From your codespace, choose **Open in VS Code**.
4. Confirm your remote workspace opens.

## Checklist
- [ ] VS Code is installed locally.
- [ ] I opened my codespace in local VS Code.
- [ ] I can edit files and see changes in Source Control.
