# Step 8: Deploy with GitHub Pages

## Goal
Publish your site to a live public URL.

## Instructions
1. Open your repo **Settings**.
2. Open **Pages** in the sidebar.
3. Set source to deploy from branch `main` and folder `/root` (or `/docs` if your project uses docs output).
4. Save and wait for deployment.
5. Open the generated URL.

## Checklist
- [ ] I enabled GitHub Pages in my repo.
- [ ] I can open my live URL.
- [ ] My latest changes appear on the live site.
