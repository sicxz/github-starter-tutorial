# Step 4: Launch a Codespace

## Goal
Start a cloud development environment with no local install required.

## Instructions
1. In your repository, click **Code**.
2. Open the **Codespaces** tab.
3. Click **Create codespace on main**.
4. Wait for the editor to load.

## Checklist
- [ ] I launched a codespace from my repo.
- [ ] I can see my project files in the editor.
- [ ] I can edit files in the codespace.
