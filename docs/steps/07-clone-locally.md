# Step 7: Clone the Repo to Your Computer

## Goal
Create a local copy of your repository so you can work offline and use terminal Git commands.

## Instructions
1. In your repo, click **Code** and copy the HTTPS URL.
2. Open a terminal on your computer.
3. Run: `git clone <your-repo-url>`
4. Change into the project directory.
5. Open it in VS Code.

## Checklist
- [ ] I cloned my repository locally.
- [ ] I opened the local folder in VS Code.
- [ ] I can run `git status` in the project directory.
