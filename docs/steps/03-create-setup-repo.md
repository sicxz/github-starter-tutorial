# Step 3: Create Your Setup Repository

## Goal
Create a clean GitHub repository with a README that all tools will use.

## Instructions
1. In GitHub, click **New repository**.
2. Name it `project-setup` (or `project-setup-yourname`).
3. Set visibility to **Public** (recommended for easier sharing/help).
4. Check **Add a README file**.
5. Click **Create repository**.
6. Copy the HTTPS clone URL from the green **Code** button in this new repo.

## Checklist
- [ ] I created my setup repository with a README.
- [ ] The repository is under my GitHub account.
- [ ] I copied the HTTPS clone URL.
