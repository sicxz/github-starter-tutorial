# Step 6: Link Codex to Your Repo Folder

## Goal
Open the same local `project-setup` folder in Codex.

## Instructions
1. Open Codex (Desktop or CLI).
2. Open/select your local `project-setup` folder as the working directory.
3. Ask Codex to list files in the folder to confirm access.
4. Verify Codex can see `README.md` and the `.git` repository.

## Checklist
- [ ] Codex is open and authenticated.
- [ ] Codex is pointed at my local `project-setup` folder.
- [ ] Codex can read files from the repo.
