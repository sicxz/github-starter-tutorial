# Step 6: Install Essential VS Code Extensions

## Goal
Set up core extensions for HTML/CSS and Git productivity.

## Instructions
1. Open the Extensions panel in VS Code.
2. Install the recommended extensions from `.vscode/extensions.json`.
3. Reload the window if prompted.

## Checklist
- [ ] I opened the Extensions panel.
- [ ] I installed recommended extensions.
- [ ] My editor is ready for HTML/CSS and Git work.
