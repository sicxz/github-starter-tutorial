# Step 4: Clone Your Repo Locally

## Goal
Clone your setup repo so one local folder can be used by all tools.

## Instructions
1. Open GitHub Desktop.
2. Click **File** -> **Clone repository...**.
3. Select/paste your `project-setup` repository.
4. Choose a local folder location and clone.
5. Open the cloned folder in VS Code.
6. Confirm the folder contains `.git` and `README.md`.

Alternative (terminal):
```bash
git clone <your-repo-url>
cd project-setup
```

## Checklist
- [ ] I cloned my setup repository locally.
- [ ] I opened the local folder in VS Code.
- [ ] I can run `git status` in the project folder.
